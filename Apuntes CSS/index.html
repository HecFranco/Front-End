Selectores

Selector universal -

Se utiliza para seleccionar todos los elementos de la página.

* {
	margin: 0;
	padding: 0;
}

Selector de tipo o etiqueta -

Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector.

h1 { color: red; }
h2 { color: red; }

ó en este caso, CSS permite agrupar todas las reglas individuales en una sola regla con un selector múltiple. 

h1,h2 { color: red; }

Selector descendente -

Selecciona los elementos que se encuentran dentro de otros elementos. 

p span { color: red; } 
	
<p>
	...
	<span>texto1</span> 
	...
	<a href="">...<span>texto2</span></a>
	...
</p>

Selector de clase -

Selecciona los elementos que poseen esa misma clase. 

.destacado { color: red; }

afectaría a todos los elementos con clase .destacado

<body>
	<p class="destacado">Lorem ipsum dolor sit amet...</p>
	<p>Nunc sed lacus et <a href="#" class="destacado">est adipiscing<a> accumsan...</p>
	<p>Class aptent taciti <em class="destacado">sociosqu ad</em> litora...</p>
</body>

Selector de clase + de tipo o etiqueta -

Selecciona las etiquetas que a su vez poseen esa misma clase. 

p.destacado { color: red; }

sólo afectaría al primer parrafo. 

<body>
	<p class="destacado">Lorem ipsum dolor sit amet...</p>
	<p>Nunc sed lacus et <a href="#" class="destacado">est adipiscing<a> accumsan...</p>
	<p>Class aptent taciti <em class="destacado">sociosqu ad</em> litora...</p>
</body>

/* Todos los elementos de tipo "p" con atributo class="aviso" */
p.aviso { ... }
 
/* Todos los elementos con atributo class="aviso" que estén dentro
   de cualquier elemento de tipo "p" */
p .aviso { ... }
 
/* Todos los elementos "p" de la página y todos los elementos con
   atributo class="aviso" de la página */
p, .aviso { ... }
